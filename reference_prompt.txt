- 촬영 환경: 구형 아이폰 5s, 일반 세단 뒷좌석, 달리는 차량 내부, 보정 없는 원본 느낌, 해질 무렵 자연광
노란빛/주황빛 완전 금지, 차가운 자연광만 유지
오른쪽 창문에서 들어오는 부드럽게 퍼지는 역광, 실내광 없음
차 내부는 좀 어두워야 함.
플래시가 너무 심하면 실패.

- 배경 (차량 구조 완전 고정):
 인물 바로 뒤 = 하나의 측면 창문만 보임
 세단 구조만 허용
 뒷좌석 뒤 추가 창문 절대 금지
 트렁크 공간 / 후방 유리 / 3열 구조 절대 금지
 SUV / 밴 구조 완전 금지
 창문 밖은 밝게 날아가고 흐릿한 모션 블러
 배경 선명도 금지 / HDR 느낌 금지
운전석 뒷자리에 앉아서 찍은 위치여야함

- 카메라:
반대편 좌석에서 촬영
반대편에서 친구가 자연스럽게 찍어준 느낌
인물과 너무 가까우면 실패. 머리크기가 작게 나오도록 적당한거리 유지

- 프레임, 구도 :
가슴 위 ~ 배꼽 "위"까지만
배꼽 아래 절대 금지 (골반/허벅지/다리 나오면 실패)

- 방향 (완전 고정):
 인물은 왼쪽 뒷좌석(운전석 뒤)
 왼쪽 어깨 = 창문쪽
 오른쪽 어깨 = 카메라쪽
 반대로 나오면 실패

- 자세:
 등을 좌석에 기대고
 눕듯이 퍼지는 자세 금지
 몸을 살짝 안쪽으로 말면서 자연스럽게 웃으며 꼬인 상태
 허리 라인 살아있게
승모근이 없어 보이고 목을 길게 빼는 예쁜 자세, 가녀린 직각어깨
팔꿈치는 인물의 배,허리 부분에 위치하도록 고정한다 (다른위치에 있으면 실패)
뒷좌석 시트에서 너무 떨어져 있으면 실패
창문과 뒷좌석 어딘가에 냅다 앉아있는 것처럼 나오면 실패

- 고개:
 왼쪽(창문쪽)으로 확실하게 기울임
 턱 살짝 당김
 목 길어보이고 승모근 안 보이게
 몸이랑 같이 누워버리는 자세 금지

- 팔/손 (핵심 해결 구조):
 한쪽 손만 사용
 팔은 프레임 아래쪽에 위치하며 거의 보이지 않음
 팔꿈치 위치가 드러나지 않음
 팔이 좌석/창문에 기대지 않음
 팔이 위로 들리는 구조 금지
팔꿈치 위치는 배앞에 있어야 함
팔꿈치가 몸이랑 수직으로 굽혀져 있으면 실패.
팔꿈치 높이가 사람의 명치 이상으로 올라가면 실패.

- 손 (형태 완전 고정):
 주먹을 살짝 쥔 상태
 주먹 말아쥔 상태에서 손가락 있는 방향으로 코 + 입을 가림 (코/입이 보이면 실패)
 손 펼침 / 손 가로 방향 사용 절대 금지
 손목 꺾임 없음
손등으로 가리면 실패
주먹 말아쥔 상태에서 손등쪽으로 코랑 입 가리면 실패
손가락이 코랑 입 방향에 있어야 함

- 손 디테일:
 가디건 소매가 손끝까지 완전히 덮음
 손가락 / 손끝 / 손등과 같이 살이 보이면 실패

- 얼굴:
 코 + 입 완전히 가림
 눈만 보임

- 표정:
 카메라 보며 자연스러운 눈웃음
 찡그림 금지
 "예쁜척 + 수줍은 느낌" 고정

- 의상 (완전 수정):
 흰색 얇은 스퀘어넥 끈나시 (일자 넥라인)
 골반까지 내려오는 긴 기장
 배 전체 완전히 덮음
 크롭탑 절대 금지 / 배꼽 노출 절대 금지
나시가 짧으면 안됨
 회색 얇고 루즈한 가디건
 너무 두껍지 않음
 매우 아방한 핏 (옆으로 여유)
 기장은 짧지 않게 자연스럽게 떨어짐
 한쪽 어깨 자연스럽게 흘러내림
아주 얇은 끈나시여야 함. 끈이 조금이라도 두꺼우면 실패
나시가 원피스처럼 나오면 실패.

- 악세서리:
 완전 없음 (귀걸이, 에어팟, 목걸이 등 전부 금지)

- 색감 (중요 안정화):
 차가운 중립톤

 저채도 저대비
너무 선명하고 쨍한 느낌이면 실패.
 부드러운 haze + glow + blur

 HDR / 샤픈 금지

아주 살짝 플래시 터트린 느낌. 얼굴이 하얗게 뽀용해져야 함

블러리하고 살짝 글로우하게 퍼지는 느낌이 있음
